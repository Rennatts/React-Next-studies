# Next.js + React studies

Personal workspace for learning **Next.js** (App Router) and **React** with topic folders, short demos, and markdown notes.

## Quick start

```bash
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) for the study index and links to live examples.

## Repository layout

- **`docs/ABOUT_THIS_PROJECT.md`** — Purpose, conventions, and how to add new topics (useful for you and for coding assistants).
- **`src/studies/`** — One folder per topic: `README.md` plus example code.
- **`src/app/studies/...`** — Optional routes that render those examples.

## First topic

**Design patterns → Container components:** code in `src/studies/design-patterns/container-components/`, demo at `/studies/design-patterns/container-components`.
